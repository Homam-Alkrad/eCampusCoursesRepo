Q3
Find and explain the error(s), if any in the following statements: struct abc { int x; float y; }; abc s1 = {10}; abc s2 = {10}; if (s1 == s2) cout << "they are equal ..." << endl;
üí° Answer: 
Error: Cannot compare structs directly using == operator.

Explanation:
Structs cannot be compared using == operator in C++.
You must compare individual members:

if (s1.x == s2.x && s1.y == s2.y)
    cout << "they are equal ..." << endl;

Q6
Write the needed code to search a nonempty single linked list named "List" to check whether it has data = 10 or not; if it has data = 10, print "Found"; otherwise, print "Not found."
üí° Answer: 
Node* current = List;
bool found = false;

while (current != NULL) {
    if (current->data == 10) {
        found = true;
        break;
    }
    current = current->next;
}

if (found)
    cout << "Found" << endl;
else
    cout << "Not found" << endl;
Q7
Write the needed code to calculate and print the sum of all elements in a non-empty singly linked list named "List".
üí° Answer: 
int sum = 0;
Node* current = List;

while (current != NULL) {
    sum += current->data;
    current = current->next;
}

cout << "Sum = " << sum << endl;

Q11
Given a single linked-list named L that has at least two nodes in it, write the needed code to split it into two sub-lists named L1 and L2 ‚Äî one for the front half (L1), and the other for the back half (L2). If the number of elements in L is odd, the extra element should go in the front list (L1). For example if the linked list (L) = {2, 3, 5, 7, 11} then L1= {2, 3, 5} and L2 = {7, 11}.
üí° Answer: 
int count = 0;
Node* current = L;
while (current != NULL) {
    count++;
    current = current->next;
}

int mid = (count + 1) / 2;

L1 = L;
current = L;
for (int i = 1; i < mid; i++) {
    current = current->next;
}

L2 = current->next;
current->next = NULL;
Q12
Given two singly linked-lists named L1 and L2, write the needed code to merge them into a single linked-list named L. The merged list should contain all elements from both L1 and L2. For example, if the linked list L1 = {2, 3, 5} and L2 = {7, 11} then after merging: L = {2, 3, 5, 7, 11}.
üí° Answer: 
L = L1;

Node* current = L1;
while (current->next != NULL) {
    current = current->next;
}

current->next = L2;
